# HomeScore

Home inspection lifecycle scoring tool that helps homeowners understand the
condition, remaining lifespan, and replacement costs of major home components.

## Key Features
- Interactive graph visualizations of component lifecycles
- Card-based UI for component status
- Probability engine for lifecycle predictions
- CSV export of component data
